Itinerario Formativo del Certificado IFCD0110

📜 ¿Qué es un Certificado de Profesionalidad?

Es una acreditación oficial que reconoce que una persona está capacitada para desempeñar una actividad profesional específica. Está regulado por el SEPE y el Ministerio de Educación.

Para obtener el certificado hay que demostrar que realmente se han adquirido las unidades de competencia asociadas (lo que sabes hacer en el entorno profesional). En Andalucía, la solicitud y acreditación se canaliza a través del programa Andalucía Acredita.

🧱 ¿Qué es una Unidad de Competencia (UC)?

Es el conjunto de tareas y funciones que una persona debe saber hacer en el ámbito profesional. Cada UC tiene un código, un nivel y criterios de realización.

En nuestro certificado, las unidades de competencia son:

  • UC0950_2 — Construir páginas web.
  • UC0951_2 — Integrar componentes software en páginas web.
  • UC0952_2 — Publicar páginas web.

Las UC son el objetivo profesional del certificado: lo que debes demostrar que sabes hacer.

Es importante distinguir entre unidad de competencia, módulo formativo y unidad formativa.

📦 ¿Qué es un Módulo Formativo (MF)?

Es la parte del certificado que enseña los contenidos necesarios para adquirir una unidad de competencia. Puede estar dividido en varias unidades formativas.

En nuestro certificado hay tres módulos formativos:

  • MF0950_2 — Construcción de páginas web (210 h)
  • MF0951_2 — Integración de componentes software en páginas web (180 h)
  • MF0952_2 — Publicación de páginas web (90 h)

📘 ¿Qué es una Unidad Formativa (UF)?

Es un bloque temático que forma parte de un módulo formativo y organiza el aprendizaje por partes. Al superar todas las UF, se completa el módulo; y al completar el módulo, se acredita la UC vinculada.

Unidades formativas del MF0950_2:

  • UF1302 — Creación de páginas web con lenguaje de marcas (HTML) — 80 h
  • UF1303 — Elaboración de hojas de estilo (CSS) — 70 h
  • UF1304 — Plantillas y gestión de formularios — 60 h

🧱 Diferencia entre Unidad de Competencia y Módulo Formativo

Ten claro qué vas a aprender y por qué:

✅ Unidad de Competencia (UC)

  • Define lo que una persona debe saber hacer en un entorno profesional.
  • Está ligada a una ocupación concreta (p. ej., construir páginas web).
  • Es lo que se evalúa oficialmente para obtener el certificado.
  • Ejemplo: UC0950_2 — Construir páginas web.

📚 Módulo Formativo (MF)

  • Es el bloque de contenidos que enseña cómo lograr esa competencia.
  • Incluye teoría, práctica, actividades y evaluación.
  • Puede dividirse en unidades formativas más pequeñas.
  • Ejemplo: MF0950_2 — Construcción de páginas web.

👉 Resumen: La Unidad de Competencia es el destino profesional. El Módulo Formativo es el camino para llegar.

🔍 ¿Cómo se relaciona todo esto?

El Certificado de Profesionalidad IFCD0110 se compone de tres unidades de competencia, desarrolladas a través de módulos formativos, que a su vez se dividen en unidades formativas. Al completar este itinerario, obtendrás una acreditación profesional reconocida oficialmente.
