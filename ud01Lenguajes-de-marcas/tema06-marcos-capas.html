🔲 Marcos y capas en diseño web

📚 Introducción

Control estructural y visual en el diseño web

En los orígenes de la web se usaron marcos (frames) para dividir la ventana del navegador en zonas independientes. Hoy están obsoletos en favor de HTML semántico y CSS moderno (<div>, <section>, Flexbox, Grid), pero entenderlos ayuda a comprender la evolución del diseño web.

📌 1) Marcos (frames)

Permitían cargar distintos documentos HTML a la vez en una misma ventana. Se definían con <frameset> (en lugar de <body>) y varios <frame>.

📖 Leer más

Ejemplo típico: un marco izquierdo con el menú y otro a la derecha con el contenido. Presentaban problemas de accesibilidad, SEO, historial y mantenimiento, por lo que quedaron obsoletos en HTML5.

🧰 2) Creación de marcos (histórico)

<frameset cols="30%, 70%">
  <frame src="menu.html" name="menu">
  <frame src="contenido.html" name="principal">
</frameset>

Nota: <frameset> y <frame> están obsoletos desde HTML5.

🧩 3) Tipos de marcos

  • Verticales: dividen por columnas con cols.
  • Horizontales: dividen por filas con rows.
  • Anidados: combinan ambos.

📍 4) Situación y configuración

Los marcos se declaraban en un archivo “contenedor” con <frameset>. Cada <frame> podía tener src, name, scrolling, noresize, frameborder… 

<frame src="menu.html" name="menu" scrolling="no" noresize frameborder="0">
Resumen: Los marcos quedaron atrás. Hoy usamos <iframe> (puntualmente) y, sobre todo, maquetación con CSS (Flexbox/Grid) y aplicaciones SPA.

🧱 5) Capas (layers)

“Capa” es un concepto visual: un contenedor (<div>, <section>…) posicionado con CSS que puede superponerse, ocultarse o animarse.

🧾 6) Definición de capas

<div class="capa1">Capa posicionada</div>

<style>
  .capa1 {
    position: absolute;
    top: 100px; left: 50px;
    width: 200px; padding: 10px;
    background: #dbeafe; /* azul claro */
  }
</style>

🛠️ 7) Posicionamiento y apilado

  • static: flujo normal.
  • relative: relativo a su posición original.
  • absolute: respecto a su contenedor posicionado.
  • fixed: fijo en pantalla al hacer scroll.
  • z-index: orden de apilamiento (qué queda por encima).
.fondo   { position:absolute; top:50px; left:0;   z-index:1; background:#e5e7eb; }
.superior{ position:absolute; top:80px; left:30px; z-index:2; background:#fed7aa; }
🧪 Ejemplo rápido (mostrar/ocultar capa) 
<button onclick="document.getElementById('capa').classList.toggle('visible')">Ver mensaje</button>
<div id="capa" class="capa-popup">¡Hola! Soy una capa emergente.</div>

<style>
  .capa-popup{display:none; position:fixed; top:100px; left:30px; padding:20px; background:#ffefbd; z-index:10}
  .capa-popup.visible{display:block}
</style>

📈 Evolución y futuro

Pasamos de marcos rígidos a capas flexibles con CSS, componentes y SPA. Esto mejora accesibilidad, mantenimiento y rendimiento.

🎯 Conclusión

Conocer los marcos ayuda a entender el pasado; dominar capas y CSS moderno es esencial hoy.
