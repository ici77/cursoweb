🔗 1.5. Enlaces y direccionamientos

¿Qué es un enlace?

Un enlace (hipervínculo) en HTML conecta un contenido con otro: puede llevar a otra página, un documento, una imagen, una sección de la misma página o acciones como enviar un correo.

Se crea con la etiqueta <a> (de anchor, ancla). Esta etiqueta puede envolver texto, imágenes u otros elementos.

📌 Sintaxis básica

<a href="https://ejemplo.com">Texto del enlace</a>
  • href: atributo obligatorio con la dirección de destino (URL).
  • El contenido entre <a> y </a> es lo clicable.

💡 Ejemplo

<a href="https://www.w3schools.com">Ir a W3Schools</a>
📚 Saber más sobre enlaces

Información ampliada

Los enlaces sustentan la navegación web. Un sitio bien enlazado mejora la experiencia, accesibilidad y posicionamiento. Úsalos con textos descriptivos y estructura coherente (menús, botones, CTAs).

🔍 Tipos de enlace en HTML

Según el destino en href, un enlace puede ser:

Nombre Descripción Ejemplo
🟢 Externo Fuera del sitio web actual. <a href="https://www.wikipedia.org">Ir a Wikipedia</a>
🟡 Interno Otra página del mismo proyecto. <a href="contacto.html">Contáctanos</a>
🔵 Relativo Ruta relativa al archivo actual. <a href="../imagenes/logo.png">Ver logo</a>
🔴 Absoluto URL completa (http/https). <a href="https://ejemplo.com/blog/articulo.html">Leer artículo</a>
🟣 Email Abre el cliente de correo. <a href="mailto:info@ejemplo.com">Escríbenos</a>
🟠 Teléfono Llamada (móvil). <a href="tel:+34900111222">Llamar al 900 111 222</a>

📍 Anclas: Navegar dentro de la misma página

Las anclas desplazan a una parte concreta de la página sin recargarla. Útiles en documentos largos, FAQ o índices internos.

🔧 ¿Cómo se crean?

  1. Asigna un id al destino: 
    <h2 id="contacto">Contáctanos</h2>
  2. Crea el enlace con # + id: 
    <a href="#contacto">Ir a la sección de contacto</a>

🔄 Enlace vs. ancla

TipoComportamientoEjemplo
Enlace clásico Abre otra página <a href="contacto.html">
Ancla Desplaza dentro de la misma página <a href="#contacto">
📖 Buenas prácticas con anclas
  • id único en toda la página.
  • Texto del enlace claro y descriptivo.
  • Mejora la accesibilidad y la navegación por teclado.
🧭 Saber más sobre navegación interna

Ventajas

  • Navegación rápida sin recarga.
  • Índices y menús internos.
  • Mejor UX en páginas extensas.

🌐 Vínculos y experiencia de usuario

  • Claros y descriptivos: indica el destino.
  • Accesibles: buen contraste, no solo color.
  • Consistentes: estilo uniforme.

❌ Mal ejemplo

<a href="doc.pdf">Haz clic aquí</a>

✅ Buen ejemplo

<a href="doc.pdf">Descargar el manual en PDF</a>
📖 ¿Dónde se abre un enlace? (target)

🔧 target controla cómo se abre

_self (por defecto)

En la misma pestaña. Recomendado para enlaces internos.

<a href="pagina2.html">Ir a la página 2</a>

_blank

Nueva pestaña. Útil para externos, PDFs, formularios…

<a href="https://google.com" target="_blank">Buscar en Google</a>

🛡️ Seguridad con rel="noopener noreferrer"

Evita ataques como tabnabbing.

<a href="https://openai.com" target="_blank" rel="noopener noreferrer">Visitar OpenAI</a>

📋 Resumen

ValorComportamientoUso recomendado
_selfMisma pestañaInternos
_blankNueva pestañaExternos / recursos
_parentMarco padreCon iframes
_topRompe iframeContenido embebido

🎁 Bonus: Enlaces y SEO

Los enlaces afectan a la estructura, navegabilidad y autoridad de un sitio. Bien usados, mejoran la visibilidad en buscadores.

TipoEjemplo¿Aporta SEO?
Interno<a href="nosotros.html">✅ Sí
Externo<a href="https://w3.org">✅ Sí
Roto<a href="404.html">❌ No
Ancla (#)<a href="#inicio">Neutro
📖 Buenas prácticas SEO
  • Usa enlaces internos para guiar y distribuir autoridad.
  • Evita enlaces rotos.
  • Evita textos genéricos como “haz clic aquí”.
  • Usa rel="nofollow" en enlaces patrocinados o poco fiables.
<a href="https://publicidad.com" rel="nofollow">Publicidad</a>
