UF1302: Creación de páginas web con el lenguaje de marcas (80 horas)

Aprende paso a paso a construir páginas web accesibles, estructuradas y profesionales, usando HTML, multimedia y herramientas modernas.

📚 UF1302 CREACIÓN DE PÁGINAS WEB CON EL LENGUAJE DE MARCAS (80 horas)

La unidad formativa UF1302 forma parte del módulo MF0950_2 – Construcción de páginas web, enmarcado en la unidad de competencia UC0950_2. Su objetivo es proporcionar una base sólida en el desarrollo de sitios web mediante una formación estructurada en cuatro unidades didácticas.

A lo largo de estas unidades trabajarás contenidos esenciales como los lenguajes de marcas, el uso de elementos multimedia, los principios de accesibilidad y usabilidad, así como las herramientas de edición web. Cada unidad combina teoría y práctica con el fin de facilitar una comprensión progresiva y aplicada del desarrollo web.

🎯 Objetivos de la unidad formativa

  • Reconocer las características de los lenguajes de marcas
  • Comprender la estructura de un documento HTML
  • Utilizar etiquetas HTML para estructurar y dar formato a contenidos
  • Insertar enlaces, imágenes y contenido multimedia
  • Aplicar principios básicos de accesibilidad y usabilidad web
  • Instalar y utilizar herramientas de edición web (como VS Code)
