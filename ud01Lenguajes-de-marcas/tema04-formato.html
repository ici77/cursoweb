📝 1.4 Marcas para dar formato al documento

En HTML, las marcas para dar formato son etiquetas que aportan estructura y significado al texto. Aunque el estilo visual lo gestiona principalmente CSS, HTML ofrece marcas semánticas esenciales para una buena accesibilidad y SEO.

Estas marcas definen el inicio y el final del contenido afectado y pueden actuar sobre texto, párrafos o listas.

❓ ¿Qué son las marcas de formato?

¿Qué son las marcas de formato?

Son etiquetas HTML que estructuran y enfatizan partes del texto. El aspecto visual se recomienda resolver con CSS, pero la semántica viene dada por etiquetas como <strong>, <em>, <blockquote>, etc.

  • 💡 Aportan significado al contenido.
  • 📚 Clave para accesibilidad y SEO.
  • 🔧 Etiquetas obsoletas como <font> fueron reemplazadas por CSS moderno.
📖 Leer más sobre tipos de marcas

🔹 Marcas de inicio y final

La mayoría de etiquetas tienen apertura y cierre (p. ej. <b>...</b>).

🎨 Marcas de aspecto (énfasis semántico)

  • <strong> / <b>: énfasis o negrita.
  • <em> / <i>: énfasis o cursiva.
  • <u>: subrayado.
  • <mark>: resaltado.

📄 Marcas de párrafo

  • <p>: párrafo.
  • <br>: salto de línea (vacía, sin cierre).
  • <blockquote>: cita destacada en bloque.

🔤 Fuentes y colores

  • <font> (obsoleta) → usar CSS.
  • Para destacar sin semántica, usar <span> con clases CSS.

🔢 Listas

  • <ol>: lista ordenada (números).
  • <ul>: lista sin orden (viñetas).
  • <li>: ítem de lista.

✅ Ejemplo

<p>Texto con <strong>énfasis</strong> y un <br> salto de línea.</p>

<blockquote>Cita en bloque con significado.</blockquote>

<ol>
  <li>Primer elemento</li>
  <li>Segundo elemento</li>
</ol>

🔎 Accesibilidad: Marcas que ayudan a todos

Un marcado correcto facilita la comprensión del contenido por lectores de pantalla y mejora la navegación por teclado. Usa etiquetas semánticas y atributos como alt, aria-label o role.

📚 Saber más sobre accesibilidad

Accesibilidad y etiquetas HTML

Una jerarquía correcta de encabezados y formularios bien asociados es clave.

📖 Etiquetas accesibles útiles
  • <img alt="..."> – Describe la imagen. 
    <img src="logo.png" alt="Logo de la empresa">
  • <label for="id"> – Asocia controles de formulario. 
    <label for="email">Correo:</label>
<input type="email" id="email" name="email">
  • Encabezados jerárquicos – <h1>..<h6>. 
    <h1>Sección principal</h1>
<h2>Subsección</h2>
<h3>Detalle</h3>

🟡 Bonus: Cómo ver las marcas en el navegador

El inspector del navegador (Chrome, Firefox, Edge…) permite ver y probar el HTML/CSS real de cualquier página en tiempo real.

🔍 ¿Qué es el inspector?

Ideal para aprender, depurar y experimentar. Los cambios no se guardan al recargar.

📖 Curiosidades útiles
  • Editar etiquetas y ver cambios al instante.
  • Probar estilos CSS al vuelo.
  • Simular vistas móvil/tablet/escritorio.
  • Detectar errores en el código.

🧪 Aprendemos haciendo: explora y edita HTML y CSS en vivo

🎯 Paso a paso

🔧 Abrir el inspector

  1. Abre cualquier página web.
  2. Clic derecho → Inspeccionar.
  3. Explora el panel de HTML y CSS.

✨ Editar etiquetas HTML

  • En “Elements”, localiza una etiqueta y edítala (doble clic).

🎨 Cambiar estilos CSS

  • Prueba propiedades como color, font-size o border.
